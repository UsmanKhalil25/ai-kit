# Find Skills

Tool for discovering and installing skills from the skills.sh ecosystem.

## Quick Start

1. Copy this package to your project:
   ```bash
   cp -r find-skills/* /path/to/your/project/.opencode/
   ```

2. Use the skill to search for available skills

## Commands

- `npx skills find [query]` - Search for skills
- `npx skills add <package>` - Install a skill
- `npx skills check` - Check for updates
- `npx skills update` - Update all skills

## Browse Skills

Visit https://skills.sh/ to explore available skills.

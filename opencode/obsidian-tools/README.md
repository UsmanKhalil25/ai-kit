# Obsidian Tools

Utilities for working with Obsidian vaults in OpenCode.

## Quick Start

1. Copy this package to your project:
   ```bash
   cp -r obsidian-tools/* /path/to/your/project/.opencode/
   ```

2. Use the skills in your OpenCode sessions

## What's Included

- **obsidian-cli/**: Vault operations (read, create, search notes)
- **obsidian-markdown/**: Obsidian-flavored markdown formatting
- **obsidian-bases/**: Database views and formulas for notes

## Usage

Load the skills in your OpenCode configuration and use them to manage your Obsidian vault programmatically.
